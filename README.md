## Code Syntax
### if-else Code Snippet
```py
if expression1 :
  statement
else :
  statement
```

### if-elif Ladder
```py
if expression1 :
  statement1
elif expression2 :
  statement2
elif expression3 :
  statement3
else:
  statement4
```
